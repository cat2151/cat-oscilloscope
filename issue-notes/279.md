# issue メイン波形表示の赤とオレンジの位相マーカーがバグっている #279
[issues #279](https://github.com/cat2151/cat-oscilloscope/issues/279)

# PR 276を取り込んだ結果、バグった
- userの期待：
    - メイン波形エリアにて：
        - オレンジstartから、オレンジend までが、表示されていること（画面両端にオレンジが表示されること）
        - 赤startと、赤endは、画面内に表示されていること
        - 結果的に、オレンジと赤の、メイン波形エリアでの場所は、動きが少ないこと
- 実際に発生したこと：
    - 赤startが画面内に表示されていないことがある
- 課題
    - そもそも4周期バッファにおける、赤とオレンジの場所が、折れ線グラフでしかわからない
- 対策
    - 可視化。4周期バッファ表示エリア（「今回の波形」エリア）において、
    - 今は、赤とオレンジを、折れ線グラフでのみ表示しているが、
    - 赤とオレンジを縦線としても表示する
    - まずそれで可視化してから、位相マーカーバグの対策を考える
    - よって、このissueでは、位相マーカーバグの修正そのものは、禁止する
        - これまでの経験上、安易で誤った解決手段をハルシネーションしてエンバグの積み重ねによるコードベース破壊が発生、のリスクが高い、と判断する
